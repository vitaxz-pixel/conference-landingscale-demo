Tech Conference Website

This is a simple Tech Conference website built using HTML and CSS.  
The website presents information about a tech event, featured speakers, and contact details.

Features
- Header with navigation
- Conference introduction section
- Featured speakers section
- About section
- Footer with contact information

Built With
- HTML5
- CSS3

 What I Learned
While building this project, I learned:

- How to structure webpages using semantic HTML
- How to group content using section and article
- How to style layouts using CSS
- How to align elements using inline-block
- Understanding box-sizing and spacing

Future Improvements
- Make the site responsive for mobile devices
- Implement Flexbox and CSS Grid
- Improve design and typography
- Add interactivity using JavaScript
